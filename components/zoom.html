Attributes #

X position of the zoomed area, defaults to 0.

y #

Y position of the zoomed area, defaults to 0.

scale #

Zoom scale, defaults to 1.

scaleMin #

Minimum zoom scale, defaults to 0.1.

scaleMax #

Maximum zoom scale, defaults to 2.
