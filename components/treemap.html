mode #

Render mode

Type: "binary" | "squarify" | "sliceDice" | "slice" | "dice"

  • binary: Recursively partitions the specified nodes into an approximately-balanced binary tree, choosing horizontal partitioning for wide rectangles and vertical partitioning for tall rectangles

  • squarify: mplements the squarified treemap algorithm by Bruls et al., which seeks to produce rectangles of a given aspect ratio

  • sliceDice: If the specified node has odd depth, delegates to treemapSlice; otherwise delegates to treemapDice

  • slice: Divides the rectangular area specified by x0, y0, x1, y1 vertically according the value of each of the specified node’s children. The children are positioned in order, starting with the top edge (y0) of the given rectangle. If the sum of the children’s values is less than the specified node’s value (i.e., if the specified node has a non-zero internal value), the remaining empty space will be positioned on the bottom edge (y1) of the given rectangle

  • dice: Divides the rectangular area specified by x0, y0, x1, y1 horizontally according the value of each of the specified node’s children. The children are positioned in order, starting with the left edge (x0) of the given rectangle. If the sum of the children’s values is less than the specified node’s value (i.e., if the specified node has a non-zero internal value), the remaining empty space will be positioned on the right edge (x1) of the given rectangle

Default Value: binary
